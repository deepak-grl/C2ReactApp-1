Parameters

Command_TypeType of Command. 00b = REQ (Request from Initiator Port) 01b = ACK(Acknowledge Response from Responder Port) 10b = NAK(Negative Acknowledge Response fromResponder Port) 11b = BUSY(Busy Response from Responder Port)
Object_PositionFor the Enter Mode, Exit Mode and Attention Commands.
Struc_VDM_VerStructured VDM Version. Version Number of the Structured VDM . Version 1.0 = 00b Version 2.0 = 01b Reserved = 10b and 11b
SVIDUnique 16-bit unsigned integer (0xFF01 for Displayport alternate mode)
VDM_Type1 = Structured VDM. 0 = UnStructured VDM.
Returns
